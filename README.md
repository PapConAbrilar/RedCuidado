# Plataforma de Capacitación para ONG (ELEAM)

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una plataforma web de capacitación orientada a una ONG dedicada al cuidado de personas mayores en contextos ELEAM (Establecimientos de Larga Estadía para Adultos Mayores).

La plataforma permitirá gestionar usuarios por roles (Administrador, Profesor y Trabajador), crear y asignar capacitaciones tanto transversales como específicas por área, aplicar evaluaciones automatizadas y realizar seguimiento del progreso mediante herramientas de análisis y visualización de datos.

---

## 🎯 Objetivos

* Facilitar el proceso de capacitación del personal.
* Reducir la carga operativa de capacitaciones presenciales.
* Digitalizar evaluaciones y certificaciones.
* Permitir el seguimiento del desempeño por usuario y por área.

---

## ⚙️ Funcionalidades Principales

* Gestión de usuarios y roles.
* Creación y administración de capacitaciones.
* Asignación de cursos por área o de forma transversal.
* Evaluaciones tipo test con corrección automática.
* Generación de certificados.
* Dashboard con estadísticas y seguimiento de progreso.

---

## 🧠 Contexto

El sistema está diseñado considerando:

* Alta rotación de personal.
* Usuarios con bajo nivel de alfabetización digital.
* Necesidad de cumplimiento de estándares en ELEAM.
